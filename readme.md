# System zarządzania siecią hoteli

Projekt realizowany w ramach przedmiotu **Projektowanie Systemów Informatycznych**.

Celem projektu jest zaprojektowanie systemu informatycznego wspomagającego zarządzanie **siecią hoteli**, w szczególności obsługę rezerwacji pokoi, zarządzanie hotelami w sieci oraz obsługę gości.

# Opis projektu

Projekt dotyczy stworzenia koncepcji systemu informatycznego dla **sieci hoteli**, który umożliwia zarządzanie hotelami należącymi do jednej organizacji oraz obsługę procesu rezerwacji i pobytu gości.

System ma wspierać:

- zarządzanie hotelami w sieci
- zarządzanie pokojami hotelowymi
- obsługę rezerwacji
- zameldowanie i wymeldowanie gości
- przechowywanie danych klientów
- obsługę płatności za pobyt

Projekt obejmuje analizę systemu oraz przygotowanie dokumentacji projektowej.

---

# Zakres funkcjonalny systemu

## Zarządzanie hotelami

System umożliwia:

- dodawanie nowych hoteli do sieci
- przechowywanie informacji o lokalizacji hotelu
- zarządzanie dostępnością pokoi w hotelu

## Zarządzanie pokojami

System umożliwia:

- definiowanie pokoi w hotelu
- określenie typu pokoju (np. jednoosobowy, dwuosobowy, apartament)
- zarządzanie ceną oraz dostępnością pokoju

## Obsługa gości

System umożliwia:

- rejestrację danych gościa
- przeglądanie historii rezerwacji
- przypisanie gościa do rezerwacji

## Obsługa rezerwacji

System umożliwia:

- wyszukiwanie dostępnych pokoi
- tworzenie rezerwacji
- anulowanie rezerwacji
- zarządzanie statusem rezerwacji

## Obsługa pobytu

System umożliwia:

- zameldowanie gościa
- wymeldowanie gościa
- rozliczenie pobytu

W systemie można wyróżnić następujących aktorów:
